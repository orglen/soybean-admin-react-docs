## cleanup

删除目录: node_modules, dist, 等

```ts copy
sa cleanup
```

## update-pkg

更新 package.json 依赖版本

```ts copy
sa update-pkg
```

## git-commit

生成符合 Conventional Commits 标准的提交信息

```ts copy
sa git-commit
```

## git-commit-verify

验证 git 提交信息，确保符合 Conventional Commits 标准

```ts copy
sa git-commit-verify
```

## changelog

生成 changelog

```ts copy
sa changelog
```

## release

发布，更新版本，生成 changelog，提交代码

```ts copy
sa release
```

## gen-route

生成路由

```ts copy
sa gen-route
```
